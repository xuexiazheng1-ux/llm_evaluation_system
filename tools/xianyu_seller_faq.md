# 闲鱼卖家机器人客服标准问答知识库

> 本知识库专为闲鱼卖家设计，用于自动化回复买家高频咨询。内容覆盖售前、售中、售后及账号规则，支持 RAG 系统检索与自动回复配置。

---

## 一、核心说明

- **目标**：7×24 小时自动化值守，提升转化率，解放卖家时间。
- **使用方式**：可将本文件直接上传至 RAG 知识库，或按条目导入自动回复工具。
- **结构**：每个问答对包含“标准问”“相似问”“答案模板”，便于系统泛化匹配。

---

## 二、标准问答内容（按场景分类）

### 场景一：售前咨询与商品询问

#### 1. 商品基础询问
- **标准问**：这个商品还在吗？
- **相似问**：还有吗 / 出了吗 / 在不在 / 还在卖吗
- **答案模板**：
  > 亲，商品还在的，随时可以下单哦！如果有任何问题欢迎随时问我 🙌

#### 2. 商品细节咨询
- **标准问**：能看看细节图吗？
- **相似问**：实物图有吗 / 瑕疵在哪里 / 有没有划痕 / 能不能拍个视频
- **答案模板**：
  > 亲，关于您提到的商品细节，我这边已经为您准备了详细的图片/视频说明，请查看附件。如果您还有其他想了解的细节，欢迎随时提问，我会尽力帮助您解决！ 

#### 3. 尺码/规格咨询（以服饰为例）
- **标准问**：我身高 160，100 斤，穿什么码？
- **相似问**：尺码怎么选 / 这个码偏大还是偏小 / 适合多高的人穿
- **答案模板**：
  > 亲，这款商品尺码偏【标准/偏大/偏小】。根据您提供的身高体重，建议选择【S/M/L】码。如果您不确定，可以提供更详细的尺寸（胸围/腰围），我帮您参考一下~ 

#### 4. 成色/功能咨询（以数码为例）
- **标准问**：手机电池健康多少？
- **相似问**：电池效率 / 有没有拆修过 / 功能都正常吗 / 有 ID 锁吗
- **答案模板**：
  > 亲，这款手机的电池健康度是【XX%】，【无拆无修/有过轻微维修记录】，所有功能【全部正常/具体说明】。如果需要爱思助手检测图或更多测试视频，我可以发给您参考 

### 场景二：价格与交易条件

#### 5. 砍价应对
- **标准问**：能便宜点吗？
- **相似问**：最低多少 / 少一点吧 / 有优惠吗 / 包邮吗
- **答案模板**（根据降价策略选择分支）：
  - **不接受砍价**：亲，价格已经是全网最低了，暂时不讲价哦，喜欢可以直接拍下~
  - **接受小幅砍价**：亲，最低【XX元】可以出，再低真的亏本了，您看可以吗？
  - **砍价幅度过大**：亲，这个价格确实出不了。不过如果您今天拍，我可以送您一个小礼品/包邮 

#### 6. 包邮询问
- **标准问**：包邮吗？
- **相似问**：运费多少 / 邮费谁出 / 能包邮吗
- **答案模板**：
  > 亲，本店满【XX元】包邮，您选购的商品总价【自动计算】元，【已满足/还差XX元】包邮条件。如果不满足，运费是【XX元】（发【快递名称】） 

#### 7. 自提/面交咨询
- **标准问**：可以自提吗？
- **相似问**：面交 / 在哪里 / 上门取 / 同城交易
- **答案模板**：
  > 亲，支持自提的！地址在【城市-区域】，具体定位可以私信发您。自提前请提前约时间哦~ 

### 场景三：下单与发货

#### 8. 发货时间询问
- **标准问**：什么时候发货？
- **相似问**：今天能发吗 / 多久发货 / 什么时候寄出
- **答案模板**：
  > 亲，下单后【24/48】小时内发货，默认发【快递名称】。发货后我会第一时间把单号发给您，请放心~ 

#### 9. 急单催发
- **标准问**：能今天发吗？我急用
- **相似问**：加急 / 今天能发出吗 / 能不能尽快
- **答案模板**：
  > 亲，我尽量帮您今天发出！如果您现在拍下，我争取【XX点】前给您发货。感谢理解！🙏

#### 10. 改地址/改规格
- **标准问**：我要改地址
- **相似问**：地址错了怎么改 / 想换颜色 / 能换尺码吗
- **答案模板**：
  > 亲，请提供正确的【收货人+电话+地址】，我会在发货前帮您修改。如果已经发货，需要您收到后自行联系快递转寄哦~ 

### 场景四：售后服务

#### 11. 退货/退款咨询
- **标准问**：能退货吗？
- **相似问**：不想要了能退吗 / 怎么退货 / 退款流程
- **答案模板**：
  > 亲，本店支持【7天无理由退货/不支持无理由退货，仅质量问题可退】。退货需要您保证商品完好、不影响二次销售。退货运费根据【谁的责任】承担。具体流程：闲鱼 APP → 我的订单 → 申请退款 

#### 12. 质量问题处理
- **标准问**：收到有瑕疵/坏了
- **相似问**：东西是坏的 / 描述不符 / 发错货了 / 有破损
- **答案模板**：
  > 亲，非常抱歉给您带来不好的体验！请您提供【瑕疵照片/视频】，我核实后会给您处理方案（补差价/部分退款/退货退款）。闲鱼聊天记录可以作为维权证据，请放心 

#### 13. 买家未确认收货提醒（卖家自用知识）
- **标准问**：买家一直没确认收货怎么办？
- **相似问**：订单自动确认时间 / 钱什么时候到账 / 买家不确认
- **答案模板**：
  > 亲，闲鱼系统会自动确认收货：买家签收后【10天】未主动确认，系统会自动打款给您。如果买家一直不确认，耐心等待即可，资金是安全的 

### 场景五：卖家账号与规则

#### 14. 账号违规/限流（卖家自用知识）
- **标准问**：账号为什么被限流了？
- **相似问**：为什么发不了宝贝 / 违规怎么申诉 / 被禁言了
- **答案模板**：
  > 亲，账号异常通常是因为违反了《闲鱼社区交易规则》。请打开闲鱼 APP → 【我的】→ 【设置】→ 【安全中心】→ 【违规记录】查看具体原因。如有异议，可在详情页点击【申诉】 

#### 15. 如何设置自动回复（卖家自用知识）
- **标准问**：自动回复怎么设置？
- **相似问**：怎么开通 AI 回复 / 托管服务怎么用 / 自动回复教程
- **答案模板**：
  > 亲，设置自动回复有两种方式：
  > 1. **全局设置**：闲鱼 APP → 【我的】→ 【设置】→ 【全局自动回复设置】→ 编辑内容 → 开启所有宝贝自动回复
  > 2. **单品设置**：【我的发布】→ 找到商品 → 点击右侧三个点 → 【自动回复设置】→ 单独编辑
  > 此外，还可以开通闲鱼官方的“智能 AI 托管服务”，实现更智能的自动应答

## 三、知识库组织建议（供 RAG 系统优化）

### 3.1 结构化存储格式（推荐）

| 字段       | 示例                                   |
|------------|----------------------------------------|
| 标准问     | 能便宜点吗                             |
| 相似问     | 最低多少/少一点/有优惠吗/包邮吗        |
| 场景标签   | 价格谈判                               |
| 优先级     | 高                                     |
| 答案模板   | 亲，最低【XX元】可以出，再低真的亏本了 |
| 变量占位符 | 【XX元】需动态填充                      |
| 条件分支   | 如果砍价幅度 >15% → 拒绝+推优惠券       |

### 3.2 三层话术体系

- **通用话术层**：欢迎语、感谢语、结束语等（适用于所有商品）  
  > 示例：您好！感谢您的关注，很高兴为您服务哦~有什么问题欢迎随时提问 

- **商品专属层**：与具体商品属性关联的问答（如某款手机的电池、屏幕等）  
  > 示例：这款 iPhone12 的电池健康度是 92%，无拆修，功能全好

- **应急话术层**：处理投诉、纠纷等敏感场景  
  > 示例：非常抱歉给您带来不好的体验，我们正在核实您反馈的问题，会尽快给您解决方案

### 3.3 变量与动态内容

支持变量插入可大幅提升回复精准度，常见变量示例：

| 变量          | 说明                 | 示例                                 |
|---------------|----------------------|--------------------------------------|
| {商品名称}    | 自动替换为当前商品   | 您咨询的{商品名称}还在的             |
| {商品价格}    | 自动提取商品标价     | 当前价格{商品价格}元                 |
| {包邮门槛}    | 卖家设置的包邮条件   | 满{包邮门槛}元包邮                   |
| {快递公司}    | 默认快递             | 发{快递公司}，预计 3 天到达           |

## 四、优化建议

### 4.1 避免“暴躁 AI”问题
- 话术语气保持亲切友好，多用“亲”“哦~”“啦”等软化词。
- 避免使用祈使句、反问句（如“问这么多干嘛”“要就快点说”）。
- 定期检查自动回复记录，发现异常及时调整。

### 4.2 数据驱动更新
- 定期分析“未匹配”的买家问题，将高频新问题补充进知识库。
- 关注聊天记录中买家反复追问的点，优化相应话术。
- 统计不同商品的咨询热点，针对性完善商品专属话术。

### 4.3 人工兜底机制
设置明确的转人工规则，当出现以下情况时及时通知您人工介入：
- 买家情绪激动（检测到辱骂词）
- 涉及复杂纠纷（如“假货”“被骗”）
- 用户连续 3 次追问同一问题
- 用户明确要求“转人工”

## 五、使用指南

1. **上传至 RAG 系统**：将本文件直接导入您的知识库（如 Dify、FastGPT、AnythingLLM 等），系统会自动解析标题和段落。
2. **导入自动回复工具**：如果您的工具支持批量导入问答对，可按表格格式整理后导入。
3. **个性化调整**：将答案模板中的【XX】占位符替换为您实际的商品信息（价格、发货时间等）。

> **提示**：对于更精细的检索效果，建议将每个问答对拆分为独立的 Markdown 文件，并以“标准问”作为文件名。
